# Casa Telmo - Hostel en San Telmo

Sitio web oficial de Casa Telmo, un hostel auténtico en el corazón del casco histórico de Buenos Aires.

## Características
- Diseño responsive
- Información de contacto
- Galería de imágenes
- Promociones y ofertas
- Alianza con Experience Baires

## Tecnologías
- HTML5
- CSS3
- Diseño mobile-first

## Desarrollo
Para contribuir al proyecto, cloná el repositorio y abrí `index.html` en tu navegador.
